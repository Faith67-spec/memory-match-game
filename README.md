📌 Memory Match Game – Global Hack Week Submission
🎮 About the Project

A simple and interactive Memory Match Game built as part of the Global Hack Week challenge.
The goal is to match all pairs of cards in the fewest moves and shortest time possible.

This project focuses on game logic, UI interactivity, and smooth user experience using web technologies.

✨ Features
🃏 Card flip animations
🧠 Memory matching logic
🔄 Restart game functionality
⏱️ Move counter / timer (if included)
🎉 Win detection system
💡 Clean and responsive UI
🛠️ Tech Stack
HTML5
CSS3
JavaScript (Vanilla JS)

▶️ How to Run Locally
git clone (https://github.com/Faith67-spec/memory-match-game)
cd memory-match-game
open index.html

Or use Live Server in VS Code.
